
恒定刺激法测量行走时知觉到水平/竖直方向抖动的阈限

2*2*7设计
两个场景：sphere（empty背景里一个蓝球）和room
两个抖动方向：水平和竖直
7个抖动大小：0.002:0.005:0.032

步速为2.5 km/h ~ 0.7m/s
实验中被试需要注视场景中央的注视点
每个场景持续3s，场景消失后，被试立刻进行判断：抖动/不抖
被试口述判断，由主试按键：leftArrow/rightArrow

每个刺激条件共10个trial
共2*2*7*10 = 280个trial

每个trial约10s，共50min
中途随时可以休息
在主试按键之前，记下按哪个键，然后引导被试下跑步机
等休息完，被试上跑步机后，按刚才的键，继续实验

被试到球/房间最远端距离为10m
